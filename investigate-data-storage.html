How Data Storage Works in This System:

  1. mock/data.json - This is only used as initial template data when the system starts fresh
  2. localStorage - This is where all your actual data is stored in the browser
  3. When you add employees - They are saved to localStorage, NOT to data.json
  4. data.json never changes - It's a static file that can't be modified by web browsers
Important: Web browsers cannot modify files on your computer for security reasons. That's why data.json doesn't update - all changes are stored in browser localStorage instead.
